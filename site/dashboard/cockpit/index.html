系统硬控制规则

  1. 1. 未闭环质量缺陷、未通过单元工程验收或未完成重要隐蔽联合验收时，不得转入下一工序或上级验收。
  2. 2. 重大事故隐患未治理并复核销号时，关联作业面不得开工、复工或确认实物工程量完成。
  3. 3. 设计变更须先完成技术审查和规定权限审批，再同步合同价款、进度基线、概算占用与档案目录。
  4. 4. 计量支付以监理签认的合格工程量、合同条款及批准变更为依据，支付结果单向归集至投资完成额。
  5. 5. 合同完工验收和竣工验收前，应完成质量评定、专项验收、结算资料及项目档案完整性校核。
